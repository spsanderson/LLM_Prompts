Read the meeting transcript and produce a PUID document with four sections: Purpose, Updates, Issues, and Decisions.

Purpose:
- Project Name
- Description
- Business Value
- Success Criteria

Updates:
- Date
- Status Summary
- Milestone Achieved
- Next Steps
- Owner

Issues:
- ID
- Issue Description
- Impact
- Owner
- Target Resolution Date

Decisions:
- Date
- Decision Made
- Rationale / Context
- Stakeholders Involved

Rules:
- Use only transcript content.
- Do not invent facts.
- Use “Unclear” if needed.
- Use Markdown headings and tables.
- If a section has no content, write “No new updates.”
- Output only the PUID document.
